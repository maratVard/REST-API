Проект демонстрирует реализацию ключевых принципов REST.

Технологический стек:

Backend-Java 17 + Spring Boot 3.x
База данных-PostgreSQL, MongoDB
Кэширование-Redis
Контейнеризация-Docker

Функциональные возможности регистрация и аутентификация пользователей (JWT); создание, обновление и отслеживание заказов; асинхронная отправка уведомлений; централизованная конфигурация сервисов; обнаружение сервисов и балансировка нагрузки; мониторинг метрик и логирование событий.

Как начать работу:
Клонируйте репозиторий: git clone https://github.com/maratVard/IprodyMicroservicesNew.git
Соберите все сервисы: mvn clean install
Запустите зависимости через Docker Compose: docker-compose up -d
Откройте API Gateway
